# Caso de Prueba - Ventas 01

Este archivo debe contener un PROGRAMA DEMOSTRATIVO escrito en el lenguaje DSL de Ventas.
Debe evidenciar una situación real de Nancy's Collection.

El compilador (lexer + parser) procesará este archivo de código fuente DSL.

Ejemplo hipotético:
REGISTRAR cliente nombre="Maria Lopez" DNI="12345678"
VENDER producto="Vestido Floral" precio=150.00 descuento=10%
CALCULAR total
PAGAR metodo=tarjeta
GENERAR recibo
